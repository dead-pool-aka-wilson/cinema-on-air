## 5. 실패/종료된 플랫폼과 교훈

### Quibi ($1.75B → 6개월 만에 폐업, 2020)
- **실패 원인**: COVID 타이밍(모바일 전용인데 집에서 TV 시청 증가), 공유 불가(SNS 바이럴 차단), 기존 YouTube/TikTok과 경쟁, 콘텐츠 차별화 부족
- **교훈**: 모바일 전용은 위험, 공유/바이럴 기능 필수, 콘텐츠만으로는 부족 — 커뮤니티/발견 메커니즘 필요

### Withoutabox (2019 종료)
- **실패 원인**: FilmFreeway에 시장 빼앗김, Amazon 방치
- **교훈**: 사용자 경험과 지속적 개선이 핵심

### ONFIFN (2023 종료, 한국)
- **실패 원인**: 팬데믹 후 오프라인 회귀, 저작권자 온라인 동의 감소, 관객 취향(극장 선호)
- **교훈**: 영화제 전용 모델은 지속 가능성 낮음. 상시 운영 콘텐츠 필요

## 7. 시장 갭 분석 (기존 서비스에서 부족한 점)

### 크리에이터 관점
1. **진정한 아카이빙**: 대부분 플랫폼은 "스트리밍"이지 "아카이빙"이 아님. 메타데이터(출연진, 스태프, 영화제 수상 이력, 기술 사양) 체계적 관리 부재
2. **포트폴리오 기능**: 감독/배우가 자신의 작업 이력을 전시할 통합 프로필 부재
3. **수익화 다양성**: 대부분 광고 또는 구독만. 직접 후원, 라이선싱, 상영권 판매 등 복합 수익 모델 부재
4. **영화제 연동**: FilmFreeway에서 제출 → 수상 → 플랫폼 스트리밍의 연결 흐름이 끊김

### 시청자 관점
1. **발견 메커니즘**: 대부분 에디터 큐레이션 의존. 알고리즘/태그/무드/테마 기반 검색 부족
2. **단편영화 전용 UX**: 대부분 YouTube/Vimeo 임베드에 의존, 단편에 최적화된 시청 경험 부재
3. **메타데이터 인덱싱**: 감독, 배우, 장르, 런닝타임, 제작국, 영화제 수상 등으로 검색 불가
4. **커뮤니티/소셜**: 리뷰, 평점, 리스트 만들기 등 Letterboxd 같은 소셜 기능 부재

### 시장 관점
1. **한국 시장**: 무빌리지(초기), MovieBloc(블록체인 니치), 인디그라운드(공공) — 민간 범용 플랫폼 공백
2. **아카이빙 + 스트리밍 통합**: 아카이빙(KOFA 같은 기관용) ↔ 스트리밍(Cineshort 같은 소비용)이 분리됨
3. **크리에이터 프로필 경제**: 감독/배우 프로필 → 필모그래피 → 팔로우 → 신작 알림 시스템 없음

## 13. Shortverse — 가장 가까운 경쟁자 분석

**Shortverse** (Short of the Week 운영, 2022 런칭)가 현재 가장 유사한 서비스:
- 크리에이터 랜딩 페이지, 영화제/수상 트래킹, 크레딧 관리
- 비공개 스크리너, 업계 네트워킹, 피어 피드백
- **BUT**: 소비자 발견 플랫폼이 아님, 시청자 기반 제한적, 아카이빙 아님

**우리와의 차별점 가능성**:
- Shortverse = 크리에이터 도구 (포트폴리오)
- 우리 = 아카이빙 + 스트리밍 + 소셜 발견 (양면 플랫폼)

### 한계점 (우리의 기회)

| **Achievement** | 시청 배지 | "10편 시청", "아시아 10개국 작품 시청" 등 게이미피케이션 |

### Steam 알고리즘의 핵심 원리 (2026년 기준)
- **CTR (클릭률) 기반**: 캡슐 이미지 → 클릭 → 페이지 체류 → 행동 (위시리스트/구매)
- **태그 매칭**: 사용자 행동 기반 태그 프로필 ↔ 콘텐츠 태그 매칭
- **피드백 루프**: "알고리즘은 플레이어 관심에 빠르고 자동으로 반응" (Valve 공식)
- **큐레이터 시스템**: 커뮤니티 신뢰 기반 추천 (봇이 아닌 사람)
- **CTR 벤치마크**: 2% = 나쁨, 4% = 보통, 7%+ = 우수

### Steam 모델에서 가져와야 할 것
1. **양면 플랫폼 구조**: 크리에이터 도구 + 소비자 경험 동등 투자
2. **태그 기반 발견**: 에디터 큐레이션 + 알고리즘 + 커뮤니티 태그 삼위일체
3. **Discovery Queue**: 매일 개인화된 추천 큐
4. **리뷰 시스템**: 텍스트 리뷰 + 추천/비추천 (단순하지만 강력)
5. **개발자(감독) 대시보드**: 실시간 분석, 트래픽 소스, 위시리스트 추이
6. **70/30 수익 분배**: 업계 표준, 투명한 정산
7. **기획전/세일 이벤트**: 시즌별 테마 기획전으로 트래픽 급등 유도

### Steam 모델에서 피해야 할 것
1. **과도한 콘텐츠 홍수**: Steam은 하루 수십 개 출시 → 발견 어려움. 큐레이션 유지 필수
2. **리뷰 조작**: 리뷰 봇/조작 문제 → 검증된 시청 기반 리뷰만 허용
3. **알고리즘 과의존**: "알고리즘이 판매를 죽였다" (인디 개발자 불만) → 편집 큐레이션과 병행

## 16. 확정된 결정 사항

- **타겟 시장**: 한국 우선 → 글로벌 확장
- **핵심 정체성**: Steam 모델 벤치마킹 — 양면 플랫폼 (크리에이터 도구 + 시청자 발견)
- **수익 모델**: 아직 미정 — MVP 무료 시작, 데이터 보면서 판단
- **콘텐츠 품질 관리**: 소액 등록비 (Steam Direct 방식, $10-30)
- **인디그라운드 참고**: 큐레이션 품질은 배울 점, 하지만 공공기관 한계 극복이 핵심 차별점
- **아카이빙 깊이**: IMDB급 풀 크레딧 + 필모그래피 (감독의 초기작으로 스타일/톤 파악 가능)
- **플랫폼 형태**: 웹 우선 (PC/모바일 웹)
- **개발**: 직접 개발 (본인이 개발자)

### 사용자의 핵심 인사이트 (원문 기반)
- "거장 감독의 초기 단편을 보면 스타일/톤앤매너 파악에 도움" → 필모그래피 추적이 핵심
- "KAFA 스트리밍은 작품마다 개별 승인 필요 — 불편" → 셀프서브 + 상시 접근이 차별점
- "인디그라운드는 2주 교체, 이후 볼 수 없음" → 영구 아카이빙이 핵심
- "30초~1분짜리도 강한 임팩트" → 런닝타임 다양성 포용
- "한국 단편영화가 장편 축약으로 치닫는 게 아쉬움" → 순수 단편영화 문화 지지

### 추가 발견된 한국 소스
- **KAFA 스트리밍**: 한국영화아카데미 졸업작/실습작, 개별 승인 후 시청 가능
- **네이버 한예종 채널**: 한예종 영상원 작품, 네이버 TV 기반
- **영화학교 아카이브**: 전세계 영화학교 졸업작 = 막대한 미개척 콘텐츠 소스

## 17. 크리에이터 Pain Point — "올리는 사람의 수요" (핵심)

### 실제 크리에이터 증언 요약 (커뮤니티 글)
- 나름 알려진 영화과 출신, 졸업 단편 예산 1,000만원+
- 배급사 끼고 작은 독립영화제 몇 군데 상영
- **"내 인생 달라진 거 하나도 없음"**
- 성공 = 3대 영화제 or 유명 독립영화제 상영
- **상영 가능 여부 = 배급사 네임드 의존**
- 영화제 선정위원이 모든 출품작을 다 보지 않음
- **유명 배급사 영화 위주로 필터링 → 이미 상영 전적 있는 영화 재선정 → 자기강화 루프**
- 업계가 극도로 좁음 — 이름 알리기가 최대 메리트
- "이름도 없는 영화제 뽑혔다고 좋아하지마 — 어차피 잊혀지고 사장 될 영화"

### 구조적 문제 분석

```
현재 단편영화 생태계:

감독 → 배급사(게이트키퍼) → 영화제 선정위원(게이트키퍼) → 상영
                ↓                        ↓
         유명 배급사만 통과        이전 상영 전적 우선
                ↓                        ↓
         무명 감독 = 사장          자기강화 루프 형성

연간 1,505편 단편영화 제작(2023, 서울독립영화제 기준)
→ 의미 있는 배급 성공: 극소수 (5% 미만 추정)
→ 나머지 95%+는 "잊혀지고 사장"
```

### Steam 혁명과의 정확한 대응

```
게임 산업 (Steam 이전):                  단편영화 (현재):
퍼블리셔(게이트키퍼) → 유통망 접근        배급사(게이트키퍼) → 영화제 접근
유명 퍼블리셔만 매장 진열              유명 배급사만 영화제 통과
인디 개발자 = 유통 불가                무명 감독 = 사장

Steam이 한 것:                         우리 플랫폼이 할 것:
게이트키퍼 제거 (누구나 등록)           게이트키퍼 제거 (감독 직접 업로드)
Discovery로 품질 필터링              태그/알고리즘/큐레이터로 품질 부상
인디 게임 폭발적 성장                단편영화 민주화
2025년: 인디 3작이 AAA 제침         ???
```

### 이것이 정의하는 서비스의 핵심 가치

> **"배급사 없이도 감독의 영화가 관객을 만날 수 있는 곳"**
>
> — 게이트키퍼를 제거하고, 작품의 품질이 스스로 증명되는 시스템

## 19. 플랫폼 인트로 영상 (Ident) — 서울아트시네마 & 넷플릭스 참고

### 사용자 인사이트
- 서울아트시네마의 상영 전 인트로 영상에 강한 인상 — "신비로운 배경음악, 미스테리를 향해 컷마다 점점 뚫고 들어가는 시선, 빌드업이 흥미진진, 볼때마다 새롭다"
- 김성욱(서울아트시네마 수석 프로그래머, 2002~)이 만들었는지 궁금해함
- 넷플릭스도 본인 영상 짧게 넣고 영화 시작 — 같은 역할이라는 인식

### 김성욱 & 서울아트시네마
- 김성욱: 1993년 문화학교 서울에서 시작, 2002년 서울아트시네마 개관부터 수석 프로그래머
- 영화평론가이기도 함 (네오이마주 등)
- 인트로 영상의 직접 제작자인지는 미확인 — 단, 기관 정체성을 전면적으로 주도한 인물
- 서울아트시네마 현재 위치: 경향아트힐 정동 (2022~)
- 2025년 다큐멘터리 "서울아트시네마 가는 길" (오진우 감독, 63분) — 이 안에 인트로 영상 포함 가능성

### 시네마 인트로(Ident) 역사와 브랜딩 전략

| 인트로 | 제작 | 핵심 특성 |

### 핵심 원칙
1. **경험을 신호하라, 브랜드를 광고하지 마라** — "지금부터 다른 세계에 진입"
2. **짧을수록 강하다** — 스트리밍 2~3초, 극장 16초
3. **인트로 자체가 브랜드의 예술형식이 될 수 있다** — MUBI는 영화감독에게 의뢰, Pixar 램프는 단편영화
4. **테스트가 직감을 이긴다** — Netflix 1년간 수십 개 테스트

### 우리 플랫폼에의 시사점
> 단편영화 전용 플랫폼의 인트로는 **그 자체가 단편영화**여야 한다.
> - 매년/매시즌 피처드 감독에게 인트로 제작 의뢰 가능
> - 인트로 자체가 큐레이션 스테이트먼트
> - THX가 음향 품질 인증이듯, 우리 인트로는 "시네마틱 비전" 인증

## 20. 아마추어/지방 감독 유저 세그먼트 — 핵심 타겟 분석

### 실제 제작기 기반 유저 프로필

| 속성 | 내용 |
|------|------|
| 위치 | 수도권에서 멀리 떨어진 남쪽 지방 |
| 배경 | 비전공, 독학 (YouTube), 가족 있음, 연장자 |
| 예산 | 1차: 30만원, 2차: 80만원 (대부분 배우 출연료) |
| 교육 | 지역 영화학교 수강 (시나리오 8회, 편집 4회, 조명 1회, 촬영 1회) |
| 장비 | 중고 구매 → 촬영 → 재판매 (렌탈비보다 저렴), 시청자미디어센터 편집실 |
| 캐스팅 | 필름메이커스가 유일한 채널, 연기 경력 없는 배우들 |
| 장소 | 수십 군데 거절 → 시청자미디어센터만 허락. 재촬영은 자기 집 화장실을 세트로 |
| 스태프 | 아내가 유일한 현장 스태프 |
| 후반작업 | 새벽에 주차장에서 후시녹음 직접 수행 |
| 제작지원 | 영진위 등 3군데 지원 → 전부 탈락 (서류 이해 불가) |
| 영화제 | 4군데 출품, 지역영화제 탈락, 1,400편과 경쟁 |
| 핵심 인용 | "보고 쓰고 찍고 **보이자**" |

### 검증된 Pain Points (에이전트 조사 결과)

#### 1. 영화제 선정률 — 1~2.5% (절망적)
- 서울독립영화제: 1,805편 출품 → ~28편 선정 (1.5%)
- 부산국제단편영화제: 5,966편 출품 → ~60편 선정 (1%)
- 정동진독립영화제: 1,030편 → ~22편 (2.1%)
- 미쟝센단편영화제: 1,891편 (2025년 4년만 재개)
- "1,400+ 출품" = 서울독립영화제 2023년 (1,374편) 또는 2024년 (1,704편)과 일치

#### 2. 영진위 제작지원 — 아마추어에게 사실상 접근 불가
- 단편 부문: 4억 100만원 총예산, 편당 최대 3천만원, 전국 ~13편 선정
- **자격 요건**: 영화연출경력 필요, 영화제작업신고증 필요 → 첫 영화 만드는 사람은 자격 미달
- 지역 기금 (전주영상위원회: 총 4천만원, ~6편 / 경남: 6,700만원, ~5편)도 전적(前績) 요구
- **자격 증명 함정(Credentialing Trap)**: 지원을 받으려면 전적이 필요 → 전적을 쌓으려면 지원이 필요

#### 3. 시청자미디어센터 — 지방 아마추어의 실질적 생명줄
- 시청자미디어재단 직영: 전국 12개 (부산, 광주, 강원, 대전, 인천, 서울, 울산, 경기, 충북, 세종, 경남, 대구)
- 전체 지역미디어센터: 전국 63개 (2026.1 기준, 전국미디어센터협의회)
- 무료 제공: 편집실 6실, 녹음실, 1인미디어제작실, 스튜디오, 장비 대여
- 정회원 가입 → 온라인 교육 (~2시간) → 시설/장비 예약 가능
- 편집실: 월 6일×8시간, 녹음실: 월 2일×4시간

#### 4. 필름메이커스 — 캐스팅의 유일한 통로
- filmmakers.co.kr: 한국 독립영화 캐스팅 지배적 플랫폼
- 자원봉사 배우모집 게시판: 8,638+ 포스트
- **그러나 서울 중심**: 배우 풀이 서울 집중, 지방 감독의 모집은 응답률 저조
- 대안: 엑터길드(네이버 카페), OTR(연극/뮤지컬)

#### 5. 지방 영화 교육 — 서울 집중 구조
- 주요 영화과: 한예종, 중앙대, 동국대 → 전부 서울
- 지방: 강원영화학교 등 소규모 프로그램이 산발적으로 존재
- 제작기의 수강 구성 (시나리오 8회, 편집 4회, 조명/촬영 각 1회) = 장비 부족 환경의 커리큘럼

#### 6. 초저예산 제작의 구조
- 장비: 시청자미디어센터 무료 → 또는 중고 구매 후 재판매
- 장소: 무료 (시청자미디어센터, 공공 장소)
- 스태프: 가족/친구 (무료)
- **배우 출연료가 유일한 현금 비용** → 80만원 예산의 대부분
- 비교: 한국 마이크로 버짓 상업영화 (연상호 감독 실낙원) = 2억원 → 250배

### 구조적 게이트키퍼 스택

```
LAYER 1: 장비       서울=렌탈샵 풍부 / 지방=시청자미디어센터뿐
LAYER 2: 캐스팅     서울=필름메이커스+근접성 / 지방=필름메이커스만, 배우 비이동
LAYER 3: 장소       서울=영상위 지원 / 지방=콜드콜 거절, 수개월 소요
LAYER 4: 교육       서울=대학+워크숍+업계 네트워크 / 지방=희소한 지역 프로그램
LAYER 5: 펀딩       영진위=전적+법인 요구 / 지역=전적+거주 요구 / 현실=무자격
LAYER 6: 배급/검증  영화제=1~2.5% 선정률 / 인디그라운드=8.8% / 현실=통로 없음
```

### 핵심 전략적 인사이트

> **이 유저 세그먼트는 한국 영화인 중 수적으로 가장 큰 세그먼트이나,
> 공식 통계에는 존재하지 않는다.**
>
> 제작사를 등록하지 않고, 영진위 지원에 자격이 없고,
> 영화제에서 선정되지 않고, 인디그라운드에 올라가지 않는다.
> 그런데도 영화를 만든다.
>
> 이들을 **보이게 만드는** 플랫폼 = 시장 공백의 정확한 해답

## 22. 플랫폼 범위 결정 — "만든 후의 플랫폼" (확정)

### 사용자 결정 (원문)
> "결국 감독들은 배급사 눈에 띄어야 하기 때문에 스트리밍과 관객의 지표로 배급사가 감독을 발굴하게 하면서 이 상황에서 비즈니스 기회를 찾아내는게 중요하다"

### 의미
1. **코어**: 스트리밍 + 관객 지표 + 감독 프로필 → "만든 후" 배급/발견에 집중
2. **NOT 코어**: 제작 도구 (캐스팅, 장소 DB, 장비 공유)는 우리 영역 아님
3. **핵심 메커니즘**: 관객 데이터가 배급사/영화제 프로그래머의 발굴 도구가 되는 구조

### 비즈니스 기회의 축
```
기존 구조:
  감독 → (문 두드림) → 배급사 → 영화제 → 관객
  
우리 구조:
  감독 → [플랫폼 업로드] → 관객 → [지표 축적] → 배급사가 발굴하러 옴
```

### Steam/Spotify 벤치마크
- **Steam**: 위시리스트 수, 리뷰 수, CTR → 퍼블리셔가 인디 개발자 발굴 기준
- **Spotify**: 스트리밍 지표 → 레이블이 무명 아티스트 발굴
- **우리**: 조회수, 완주율, 리뷰, 큐레이터 추천 → 배급사/영화제가 감독 발굴

### B2B 비즈니스 기회 (추후 탐색)
- 배급사/영화제 프로그래머 전용 대시보드 (트렌딩 감독, 장르별 인기 등)
- 감독 스카우팅 도구 (필터링: 수상이력 없지만 관객 반응 높은 감독 등)
- 이것이 잠재적 수익 모델 중 하나

## 23. 누벨바그 갤러리 커뮤니티 분석 — 단편영화 담론 심층 조사

### 출처: 디시인사이드 누벨바그 갤러리 (누갤), 2026.02~03, 약 20개 고유 게시글

---

### 핵심 발견 1: "김병규 비판" — 한국 단편영화 생태계의 구조적 문제

평론가 김병규의 전주 심사평이 커뮤니티에서 큰 반향을 일으킴 (관련 글 4개+). 그의 핵심 비판:
- 단편영화가 **"학생영화"와 동의어**가 되어가고 있음
- 젊은 세대에 의해 독점 → 젊음만 다루는 형식
- **"청년영화제", "청년 제작지원"** 같은 제도가 이 현상을 부추김
- 결과: 안전하고 예측 가능한 영화만 양산

**커뮤니티 반응 (핵심 인용)**:
- "제작지원 심사위원과 교수들 눈치보면서 만들어야 하는 상황에서 순수한 삶의 태도를 지켜내는 것은 쉽지 않다" (#005, 조회 343)
- "어두운 소재나 캐릭터도 유쾌하고 사랑스럽게, 혹은 정면으로 응시하고 분노하면서 만들 수 있다. 그런데 눈치보면서..." (#005)
- "문제를 매우 입체적으로 봄. 그래서 소중해" (#003, 조회 235)
- "본인이 찍은 단편영화도 그 비판에서 자유로울 수 없지 않나" (#001, 조회 256)

**플랫폼 시사점**: 현재 단편영화 생태계는 **제도(제작지원+영화제)가 창작의 방향을 왜곡**하고 있음.
게이트키퍼 제거 플랫폼은 이 왜곡에서 자유로운 공간을 제공 → "눈치보지 않는 영화"가 나올 수 있음

---

### 핵심 발견 2: "개성 없다 / 양산형" — 관객의 갈증

- "요즘 한국 단편영화는 **개성이 없다**. 또래들이 가질법한 고만고만한 고민들 열거하는게 전부. 연출방식마저 단조로움. 근데 그런 영화들이 영화제 많이 감" (#007, 조회 252)
- "단편영화는 퀄이 좀 떨어져도 **개성이나 훅**이 있어야 하는데, 요즘 영화제가 뽑아주는 건 대부분이 **양산형 영화**. 매년 보는 연출, 매년 보는 소재, 매년 보는 연기" (#020, 조회 308)
- "요즘 단편영화들은 혐오니 뭐니 하면서 **겁이 너무 많아**" (#004, 조회 219)

**플랫폼 시사점**: 관객은 "완성도 높은 안전한 영화"보다 **"거칠더라도 독특한 영화"**를 원함.
영화제가 필터링하는 기준 ≠ 관객이 원하는 기준. 이 갭이 우리 플랫폼의 기회.

---

### 핵심 발견 3: "보여주고 싶다" — 직접적 수요 시그널

- "누갤애들한테 **내 단편영화 보여주고싶다**. 졸업영화제에서 틀고 그해에 같이간 여친이랑 헤어짐. 그만큼 개병신같이 만든 영화임. 3년이 지난 지금 그 영화 다시 편집이랑 음악작업 해보고있는데 뭔가 보여주고싶음" (#012, 조회 217)

**이것이 우리 플랫폼의 정확한 유즈케이스**: 졸업작을 3년 후에 재편집하고, 그걸 보여줄 곳이 없어서 커뮤니티 게시판에 올리는 사람. 이 사람에게 "영구적 집"을 제공하는 것.

### 핵심 발견 4: "누가 보냐?" — 존재론적 질문

- "단편영화 **학생들 아니면 누가 보냐?** 갓반인들이나 영화 애호가들? 아무도 안 봄. 그들만의 스몰리그임. 단편 영화에 대한 수요는 예나 지금이나 극극극소수였고 지금에서야 근근히 살아가는 건 전국에 있는 영화학과 덕분임" (#002, 조회 101)

**이 질문에 대한 우리의 답**: Steam 이전에도 "인디 게임 누가 하냐?" 했음. Discovery 메커니즘이 없었기 때문. 태그+알고리즘+큐레이터로 발견 경로를 만들면 관객이 온다.

### 종합 인사이트

#### 커뮤니티가 말하는 한국 단편영화의 구조적 모순:

```
1. 제도(영화제+제작지원)가 "안전한 영화"를 선별 → 양산형 유발
2. 관객은 "개성 있는 영화"를 원함 → 영화제 선정작에 불만
3. 감독은 "눈치보며" 만듦 → 제도에 의한 자기검열
4. 결과: 만드는 사람도, 보는 사람도, 비평하는 사람도 다 불만

→ 우리 플랫폼의 기회:
   - 게이트키퍼 없음 = 자기검열 불필요
   - 관객 직접 반응 = 영화제 심사위원이 아닌 "사람들"이 평가
   - 다양한 태그/무드 = "양산형"을 넘어서는 발견 경로
   - 관객 지표 = 배급사가 "진짜 관객 반응"을 볼 수 있음
```

#### 핵심 타겟 유저 세그먼트 재확인:

| 세그먼트 | 커뮤니티 증거 | 플랫폼 가치 |
|----------|-------------|------------|
| **영화과 학생** | #011(연출), #012(졸업작 재편집), #006(제작지원) | 졸업작의 영구적 집 + 관객 접점 |
| **아마추어/지방 감독** | 이전 제작기 글 | 유일한 배급 경로 |
| **시네필 관객** | #009(최고 단편 추천), #015(작법 학습), #018(추천 요청) | 발견 + 큐레이션 |
| **평론가/프로그래머** | #005(김병규), #003(분석) | 트렌드 파악 도구 |
| **배급사** | (간접 — 제작기 글의 배급 구조 문제) | 발굴 도구 (관객 지표 기반) |
