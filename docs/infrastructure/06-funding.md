# 자금/펀딩: 한국 독립영화/단편영화 제작 인프라

## 1. 개요

한국 독립·단편영화 제작의 자금 조달 경로는 크게 **공공 지원(중앙·지방)**, **민간 재단**, **크라우드펀딩**, **국제 펀드** 네 축으로 나뉜다. 그러나 대부분의 공적 자금은 **영화연출경력**과 **영화제작업신고증**이라는 자격 요건을 전제로 하여, 첫 영화를 만드는 신인이나 비전공 아마추어에게는 사실상 접근이 불가능한 구조다.

이것이 바로 **자격 증명 함정(Credentialing Trap)**: "지원을 받으려면 전적이 필요하고, 전적을 쌓으려면 지원이 필요한" 무한 루프이며, Cinema on Air가 해결해야 할 핵심 구조적 문제 중 하나다.

> **핵심 수치**: 영진위 단편 부문 연간 선정 ~13편 vs 서울독립영화제 연간 출품 1,700+편.
> 공적 자금은 전체 단편영화 창작자 중 극소수만 도달 가능한 자원이다.

## 2. 공공 제작지원 — 중앙 (영화진흥위원회)

영화진흥위원회(영진위, KOFIC)는 한국 영화 제작지원의 최대 공적 기관이다. 2026년 한국영화 지원 예산을 전년 대비 161억원 증액하며 대폭 확대했다.

### 2.1. 독립예술영화 제작지원 — 단편 부문

| 항목 | 내용 |
|------|------|
| **총예산** | 4억 100만원 (2025년 기준) |
| **편당 지원** | 최대 3,000만원 이내 차등 지급 |
| **선정 편수** | 전국 ~13편 |
| **대상** | 단편 실사 극영화 (40분 미만), 순제작비 5,000만원 미만 |
| **자격 요건** | ① 영화연출경력 보유 연출자 ② 영화제작업신고증 보유 법인/개인사업자 |
| **접수 시기** | 연 1회 (2025년: 2월 3일~2월 17일) |
| **제한** | 5년 내 누적 3회 이상 지원받은 경우 불가 |

**신인 접근성**: ❌ 매우 낮음. 영화연출경력 + 영화제작업신고증 필수 요구. 첫 영화를 만드는 사람은 자격 미달.

### 2.2. 독립예술영화 전체 (장편+단편+다큐)

2026년 기준 독립예술영화 지원 예산은 **총 205억원** (전년 165억원 대비 약 40억원 증액).
- 제작지원 예산 77억원, 총 60편 내외
- 장편 극영화/다큐: 연 2회 공모
- 2026년부터 개봉지원 대상을 배급사 → **배급사, 제작사, 감독(개인)** 으로 확대

### 2.3. 중예산 한국영화 제작지원

| 항목 | 내용 |
|------|------|
| **예산** | 200억원 (2026년, 전년 100억원 대비 2배) |
| **대상** | 순제작비 20억~100억원 미만 장편 |
| **편당 한도** | 순제작비 40% 또는 25억원 중 낮은 금액 |
| **선정** | 18편 내외 |
| **신인 쿼터** | 최종 선정작의 30% 이상을 신인 감독으로 선정 |

### 2.4. AI 기반 영화 제작지원 (2026 신설)

- 장편(60분 이상): 8편 내외
- 단편(10~20분 미만): 30편 내외
- 연출자 자격: 장편 1편 또는 단편 2편 이상 연출 경력 (AI/애니 포함)

### 2.5. 시나리오 공모전

- 선정 14편, 총 1억 250만원
- 대상 3,000만원 / 최우수상 2,000만원 / 우수상 1,000만원
- 자격: 대한민국 국적의 **신인** 시나리오작가 → 비교적 신인 접근 가능

### 2.6. 기획개발지원 (제작사 부문)

- 30편 내외, 총 15억원 (편당 5,000만원)
- 자격: 한국영화 개봉작(또는 OTT 영화) 제작/프로듀서 크레딧 1편 이상 보유 제작사

## 3. 공공 제작지원 — 지방 (영상위원회)

지역 영상위원회는 해당 지역 거주 또는 활동 기반의 영화인에게 제작비를 지원한다. 대부분 **거주 요건**과 일정 수준의 **전적(前績)** 을 요구한다.

### 3.1. 전주영상위원회

| 항목 | 내용 |
|------|------|
| **단편영화 제작지원** | 총 6편 내외, 편당 최대 800만원 (2025년 기준, 총 ~4,800만원) |
| **대상** | 40분 이하 단편 극영화 (애니·다큐 제외) |
| **자격** | 전북 거주자 / 전북 대학 재·휴학생 / 2년 내 전북 영화단체 활동 이력 (만 18세 이상) |
| **장편 제작지원** | 단편 1편 이상 경력, 전북 6개월 이상 거주 제작사/연출자 |
| **후반제작지원** | 전주정보문화산업진흥원 공동 운영, 음향·색보정·DCP 제작 지원 (전국 장편/단편/지역단편/DCP 5편) |

**특이사항**: 후반제작지원 작품들의 영화제 성과 우수 — <K-Number> 서울독립영화제 대상, <유림> 단편대상·청룡영화상 단편영화상 수상 등.

### 3.2. 부산영상위원회

- 2025년 제작지원: 총 3개 부문, **10작품, 10억 8천만원**
- 뉴미디어 콘텐츠: 3편, 각 8,000만원 (총 2.4억원)
- 장편다큐멘터리: 5편, 총 2.4억원 (신인 쿼터 적용)
- 단편영화 부문은 별도 운영

### 3.3. 경기콘텐츠진흥원 (경콘진)

- 2026년 경기도 독립영화 제작지원: 총 **4억 3,000만원**, 5편 내외
- 편당 최대 1.5억원 (순제작비 50% 이내)
- 자격: 경기도 제작사 또는 경기도 IP
- **경기인디시네마 프로슈머 크라우드펀딩**: 공공+크라우드펀딩 하이브리드 모델 (텀블벅 연계)

### 3.4. 경남영상위원회

- 총 6,700만원, ~5편 선정 (2025년 기준)
- 지역 거주 요건 + 전적 요구

## 4. 문화체육관광부 및 KOCCA

### 4.1. 한국콘텐츠진흥원 (KOCCA)

- **K-콘텐츠 펀딩 활성화 지원사업**: 크라우드펀딩 기반 독립예술 콘텐츠의 성장을 지원하는 정책 프로그램. 영화 <신명>이 첫 사례로 언급됨.
- **K-콘텐츠 영상물 제작지원**: 경기콘텐츠진흥원 등 지역 진흥원과 연계한 영상 콘텐츠 제작 지원 사업 운영.
- 독립영화 전용 지원 프로그램보다는 **콘텐츠 산업 전반 지원**에 초점.

### 4.2. 문화체육관광부

- **구독형 영화관람권**: 이르면 2027년 시행 예정 — 관객 저변 확대를 통한 간접 지원 모델.
- 독립영화에 대한 직접 제작비 지원보다는 **영진위를 통한 간접 지원** 구조.

## 6. 크라우드펀딩 (텀블벅, 와디즈 등)

크라우드펀딩은 공적 자금의 자격 요건 벽을 우회할 수 있는 **가장 현실적인 대안 자금원**이다.

### 6.1. 텀블벅 (tumblbug.com)

- 국내 대표 **후원형** 크라우드펀딩 플랫폼
- 영화 카테고리 내 독립영화/단편영화 프로젝트 다수 운영
- 사례: 경기인디시네마 프로슈머 펀딩 3편 전체 성공 (2025년, 232명 후원, 총 2,000만원)
- <만남의 집> 텀블벅 펀딩 430만원 달성 (목표 대비 110%)
- <바람이 전하는 말> 목표 대비 250% 달성, 약 1,000만원 모금

### 6.2. 와디즈 (wadiz.kr)

- **투자형 + 후원형** 크라우드펀딩 플랫폼
- 영화 투자형 펀딩의 대표 사례: <신명> (2024년) — 4,200명 참여, **19억원** 모금, 70만 관객 돌파
  - 투자자 수익률 약 90% 기록
- AI 기반 추천 시스템 도입으로 크리에이터-후원자 연결 강화 중

### 6.3. 크라우드펀딩의 구조적 특징

| 항목 | 텀블벅 | 와디즈 |
|------|--------|--------|
| 유형 | 후원형 | 투자형 + 후원형 |
| 전형적 목표 | 300만~1,000만원 | 1억원 이상 |
| 자격 요건 | 없음 | 투자형은 증권신고 필요 |
| 신인 접근성 | ✅ 높음 | 후원형은 높음, 투자형은 보통 |
| 성공 요건 | SNS 팬덤, 스토리텔링 | 비즈니스 모델, 투자 수익 제시 |

## 7. 국제 지원 (BIFF ACF 등)

### 7.1. BIFF 아시아시네마펀드 (Asian Cinema Fund, ACF)

- 부산국제영화제(BIFF) 산하 핵심 펀딩 프로그램
- 운영 부문: **시나리오 개발**, **후반작업(Post-Production)**, **다큐멘터리(AND Fund)**, **공동제작**
- 2025년 기록적 지원: 850건 접수 (전년 대비 23% 증가), 14편 최종 선정
  - 시나리오 개발 3편, 후반작업 4편, 다큐멘터리(AND) 7편
- 한국 및 아시아 전역 영화인 대상
- ACF 지원작의 국제 영화제 진출 실적 우수 (칸, 베를린, 베니스 등)
- 웹사이트: acf.biff.kr

### 7.2. 기타 국제 지원

- **전주국제영화제 JJ-Fund**: 전주영화제 기획전과 연계한 제작/배급 지원 [UNVERIFIED 규모]
- **부천국제판타스틱영화제 NAFF**: 장르영화 중심 프로젝트 마켓 + 제작지원
- **아시아프로젝트마켓 (APM)**: BIFF 연계 투자·배급 미팅 플랫폼 (장편 중심)

### 8.2. 자격 증명 함정(Credentialing Trap)의 실체

```
지원 받으려면 전적이 필요
         ↕
전적 쌓으려면 지원이 필요
         ↕
결론: 첫 영화 만드는 사람은 공적 자금 체계에서 완전히 배제
```

**영진위 지원 자격 무한루프**: 영화연출경력 + 영화제작업신고증 → 첫 영화를 만드는 사람은 자격 미달 → 경력을 쌓을 기회 자체가 차단.

**지역 기금도 동일 구조**: 거주 요건 + 전적(前績) 요구 → 수도권 비전공 아마추어는 이중 배제.

**현실적 우회 경로**: CJ문화재단 스토리업(신인 타겟) + 텀블벅/와디즈 크라우드펀딩(자격 무관) + 자비 제작(30~80만원 초저예산).

## 9. Pro Mode 연동 시사점

Cinema on Air Pro Mode에서 자금/펀딩 인프라를 어떻게 연동할지에 대한 전략적 시사점:

### 9.1. 자금/펀딩 가이드 페이지

- **지원금 캘린더**: 영진위, 지역 영상위원회, CJ문화재단 등 주요 공모 일정을 통합 캘린더로 제공. 마감 알림(Push) 기능.
- **자격 셀프체크**: 사용자의 경력/거주지/제작사 등록 여부를 입력하면 지원 가능한 기금 목록 자동 필터링.
- **신청 서류 가이드**: 영진위 등 공모 서류 작성 가이드 (기존 연구에서 "서류 이해 불가"로 탈락한 사례 해결).

### 9.2. 투명성 표시

- **"이 영화는 어떤 지원을 받았나?"**: 플랫폼 내 작품 상세페이지에 제작 자금 출처를 투명하게 표시.
- 관객에게는 "공적 자금 → 시민의 세금"이라는 인식 제공, 크리에이터에게는 지원 실적 아카이빙.

### 9.3. 영진위 지원 자격 안내 + 우회 전략 가이드

- **자격 로드맵**: "지금 자격이 없다면? → 이 순서로 경력을 쌓아라" 단계별 안내.
  - Step 1: 자비 제작 or 크라우드펀딩 → Step 2: 영화제 출품 → Step 3: CJ스토리업 지원 → Step 4: 영진위 지원 자격 확보
- **제작업신고증 발급 가이드**: 절차, 비용, 소요 시간 안내.

### 9.4. 크라우드펀딩 연계 (직접 운영 아님)

- 텀블벅/와디즈 프로젝트 링크 연동 (플랫폼 내 크라우드펀딩 직접 운영은 OUT 스코프)
- 크라우드펀딩 성공 전략 가이드 제공 (SNS 활용, 리워드 설계, 스토리텔링 팁)
- 경기인디시네마 사례처럼 **공공 + 크라우드펀딩 하이브리드 모델** 벤치마킹

### 9.5. 데이터 기반 매칭

- 플랫폼 내 시청 데이터·관객 반응을 기반으로, 크리에이터에게 적합한 펀딩 채널 추천.
- 예: "귀하의 작품과 유사한 장르/테마의 텀블벅 성공 사례 3건" 자동 추천.
