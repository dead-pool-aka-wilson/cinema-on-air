# 인프라 마스터 인덱스: 한국 독립영화/단편영화 제작 생태계

## 개요

Cinema on Air의 **Pro Mode** 생태계를 구성하는 7개 핵심 인프라 카테고리를 종합 정리한 문서입니다. 한국 독립영화/단편영화 제작자들이 실제로 마주하는 제작 단계별 페인 포인트와 현존하는 솔루션(렌탈 업체, 촬영 장소, 인력 네트워크, 후반작업 스튜디오, 교육 기관, 펀딩 채널, 법적/행정 지원)을 매핑합니다.

이 인프라 리서치는 **ADR-013(Pro Mode/Enjoy Mode Two-sided App)**의 근거가 되며, Cinema on Air가 단순 배급 플랫폼을 넘어 제작 생태계 전체를 연계하는 순환 플랫폼으로 진화하는 방향을 제시합니다.

## Pro Mode 연동 개요

### 1. 관객 → 제작자 전환 경로

Cinema on Air의 **Enjoy Mode**(관객)에서 영화를 감상한 사용자가 **Pro Mode**(제작자)로 전환될 때, 다음 7개 인프라 카테고리가 순차적으로 활용됩니다:

```
[Enjoy Mode: 영화 감상]
         ↓
    [Pro Mode 진입]
         ↓
[01] 장비 렌탈 → [02] 촬영 장소 → [03] 인력/캐스팅 → [04] 후반작업 → [배포]
         ↑                                                              ↓
    [05] 교육/워크숍 (병렬)                                    [결과물 아카이빙]
    [06] 자금/펀딩 (병렬)
    [07] 법적/행정 (병렬)
```

### 2. 각 카테고리별 Pro Mode 활용 시나리오

#### 01. 장비 렌탈
- **Pro Mode 역할**: 신규 제작자가 저예산으로 촬영을 시작할 수 있도록 렌탈 업체 정보 제공
- **플랫폼 연계**: 제작자 프로필에 "추천 렌탈 업체" 섹션 추가, 제휴 할인 제공

#### 02. 촬영 장소
- **Pro Mode 역할**: 로케이션 스카우팅 비용 절감을 위해 촬영지 DB 제공
- **플랫폼 연계**: 지역별 촬영 장소 필터링, 촬영 허가 정보 통합

#### 03. 인력/캐스팅
- **Pro Mode 역할**: 배우, 스태프 섭외 네트워크 제공
- **플랫폼 연계**: 제작자 간 협업 매칭, 스태프 포트폴리오 연계

#### 04. 후반작업
- **Pro Mode 역할**: 편집, 컬러, 사운드 스튜디오 정보 제공
- **플랫폼 연계**: 후반작업 견적 비교, 스튜디오 예약 시스템

#### 05. 교육/워크숍
- **Pro Mode 역할**: 제작 기술 습득 기회 제공
- **플랫폼 연계**: 영화학교 추천, 워크숍 일정 통합, 마스터클래스 영상 아카이빙

#### 06. 자금/펀딩
- **Pro Mode 역할**: 제작 자금 확보 채널 제공
- **플랫폼 연계**: 정부 지원금 정보, 크라우드펀딩 연계, 투자자 네트워크

#### 07. 법적/행정
- **Pro Mode 역할**: 계약, 저작권, 세무 정보 제공
- **플랫폼 연계**: 계약서 템플릿, 법률 자문 네트워크, 배급 계약 가이드

## 인프라 리서치 목적 및 활용 방향

### 목적

1. **생태계 폐쇄성 진단**: 한국 독립영화 제작자들이 실제로 마주하는 진입 장벽 파악
2. **플랫폼 차별화 전략**: Cinema on Air가 단순 배급 플랫폼을 넘어 제작 생태계 전체를 연계하는 가치 제안 수립
3. **Pro Mode 로드맵 수립**: 제작자 전환 경로에 필요한 인프라 연계 우선순위 결정

### 활용 방향

#### Phase 1: 정보 통합 (MVP)
- 7개 카테고리별 주요 업체/기관 정보를 Pro Mode 내 "리소스 가이드" 섹션으로 제공
- 제작자 프로필에 "내 프로젝트 진행 상황" 추적 기능 추가 (장비 예약 → 촬영 → 후반 → 배포)

#### Phase 2: 플랫폼 연계 (Growth)
- 렌탈 업체, 스튜디오, 교육 기관과 제휴 협약 체결
- 제작자 커뮤니티 내 협업 매칭 기능 (배우, 스태프, 투자자)
- 정부 지원금 정보 자동 업데이트 및 신청 가이드

#### Phase 3: 생태계 폐쇄 (Ecosystem)
- Cinema on Air 플랫폼 내 "제작 마켓플레이스" 구축
- 렌탈 업체, 스튜디오, 교육 기관의 서비스를 플랫폼 내에서 직접 예약/결제
- 제작자 → 배급 → 관객 → 신규 제작자 전환의 완전한 순환 구조 완성
