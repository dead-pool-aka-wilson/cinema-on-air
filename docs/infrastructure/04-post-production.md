# 후반작업: 한국 독립영화/단편영화 제작 인프라

## 1. 개요 및 후반작업 단계

독립영화/단편영화의 후반작업(Post-Production)은 촬영 종료 후 최종 상영본 완성까지의 전 과정을 포괄한다. 상업영화와 달리 독립영화는 예산과 인력의 제약으로 인해 감독이 직접 편집·색보정·사운드 작업을 수행하는 경우가 많다.

**후반작업 주요 단계:**

1. **편집 (Editing)** — 촬영본 정리, 시퀀스 구성, 컷 편집
2. **색보정 (Color Grading/DI)** — 색감·톤·대비 조정, 시각적 통일
3. **사운드 디자인 및 믹싱** — 대사 정리, 효과음, 앰비언스, 음악, 최종 믹싱
4. **VFX/CG** — 시각효과 합성 (독립영화에서는 최소한으로 사용)
5. **자막 제작** — 한글/영문 자막 (영화제 출품 시 필수)
6. **마스터링 및 DCP 제작** — 최종 마스터 생성, 극장 상영용 DCP 변환

> 아마추어 감독의 실태: "새벽에 주차장에서 후시녹음 직접 수행" — 한국 독립영화 제작기 중 발견된 사례로, 후반작업 인프라 접근의 어려움을 단적으로 보여준다.

## 3. 색보정 (Color Grading)

색보정은 영화의 시각적 톤과 분위기를 결정하는 핵심 후반작업이다. 독립영화에서는 무료 소프트웨어로 자체 작업하거나, 프리랜서/소규모 스튜디오에 외주를 맡기는 두 가지 경로가 존재한다.

### 3.1 무료/저비용 도구

- **DaVinci Resolve** (Blackmagic Design): 업계 최고 수준의 색보정 도구를 무료로 제공. 노드 기반 컬러 그레이딩, ACES 색관리 지원. 유료 버전(Studio, ~₩450,000 1회 구매)은 HDR 그레이딩, 노이즈 리덕션 등 추가 기능 제공.
- **FilmConvert Nitrate**: 필름 에뮬레이션 플러그인 (~$159), Premiere/Resolve 호환

## 4. 사운드 믹싱 및 음악

### 4.1 사운드 후반작업 구성요소

- **대사 정리 (Dialogue Editing)**: 동시녹음 소스 정리, 노이즈 제거
- **후시녹음 (ADR, Automated Dialogue Replacement)**: 현장 녹음 불량 시 스튜디오 재녹음
- **폴리 (Foley)**: 발소리, 옷 스치는 소리 등 효과음 직접 제작
- **사운드 디자인**: 앰비언스(공간감), 이펙트, 전환음 설계
- **음악 (Score/OST)**: 작·편곡, 라이선스 음악 선정

## 5. DCP (Digital Cinema Package) 제작

DCP는 극장 상영 및 영화제 출품에 필수적인 디지털 시네마 배급 포맷이다. DCI(Digital Cinema Initiatives) 규격에 맞춰 JPEG2000 압축, MXF 컨테이너, 암호화 패키징을 거친다.

### 5.1 DCP 주요 규격

- **해상도**: 2K (2048×1080) / 4K (4096×2160)
- **화면비**: Flat (1.85:1, 1998×1080) / Scope (2.39:1, 2048×858)
- **오디오**: 5.1ch 또는 Stereo, 48kHz/24bit WAV
- **규격**: Interop (구형, 호환성 우수) / SMPTE (신형)

### 5.2 DCP 제작 서비스 (실명)

| 서비스 | 특징 |
|---|---|
| **비엠텍 (BMTech)** | 서울. 독립영화·단편영화 DCP 변환 전문. 2K/4K Flat/Scope, Interop/SMPTE 지원. 자막 하드코딩, 프레임레이트 변환 등 부가 서비스. 문의: 02-761-8130, koh322@daum.net |
| **전주영화제작소** (공공 지원) | 후반제작지원 사업 선정 시 DCP 제작 무료 지원. 연 2회(상·하반기) 공모 |
| **DaVinci Resolve + DCP 오픈소스 도구** | DaVinci Resolve에서 마스터 출력 → OpenDCP 또는 DCP-o-matic(무료 오픈소스)으로 DCP 패키징 가능. 단, 극장 호환성 테스트 필수 |

## 6. VFX/CG (독립영화 규모)

독립영화에서 VFX는 최소한으로 사용하는 것이 원칙이다. 예산 한계로 고품질 CG가 불가능하기 때문에, 촬영으로 해결 가능한 장면은 반드시 촬영으로 처리하고 VFX는 보조 수단으로만 활용한다.

### 6.1 독립영화 VFX 주요 활용 사례

- **와이어 리무벌**: 촬영 시 보이는 지지대/와이어 제거
- **하늘 교체 (Sky Replacement)**: 촬영 당일 날씨와 다른 하늘 합성
- **화면 합성 (Compositing)**: 그린스크린 배경 교체
- **잡티 제거**: 현대물이 보이는 시대극 촬영 시 제거 작업

### 6.2 VFX 도구 (저비용)

| 도구 | 가격 | 특징 |
|---|---|---|
| **DaVinci Resolve Fusion** | 무료 | 노드 기반 VFX 합성. Resolve 내장 |
| **Adobe After Effects** | 월 ~₩30,800 | 모션그래픽·VFX 업계 표준. 트래킹, 키잉, 파티클 |
| **Blender** | 무료 (오픈소스) | 3D 모델링·렌더링·합성. 학습 곡선 있으나 무료 |
| **HitFilm** | 무료 버전 존재 | VFX 특화 편집기. 입문자용 |

> 독립영화 CG 의뢰 시 주의: "가능하면 CG를 사용하지 마세요. 독립영화에서는 CG보다 촬영이 항상 낫습니다." — 필름메이커스 커뮤니티 CG 전문가 조언 (조회수 38만+)

## 7. 자막 제작 (한/영)

영화제 출품 시 영문 자막은 거의 필수이며, 국제영화제 출품 시 다국어 자막이 요구된다.

### 7.1 자막 제작 방식

- **SRT/VTT 파일**: 소프트 자막 (영화제 온라인 스크리너용)
- **하드코딩 (Burn-in)**: 영상에 직접 박는 방식 (DCP 내장 자막 별도)
- **DCP 내장 자막**: Interop/SMPTE 규격에 맞춘 XML 기반 자막

### 7.2 자막 제작 도구 및 서비스

- **Subtitle Edit** (무료 오픈소스): SRT 자막 타이밍 편집
- **DaVinci Resolve**: 내장 자막 기능으로 직접 제작 가능
- 번역: 프리랜서 번역가 (크몽, 숨고 등 플랫폼) 활용
- DCP 자막: 비엠텍, LI TRUST 등 DCP 전문 업체에서 패키징 시 포함

2. **시청자미디어센터 적극 활용**: 편집실(월 48시간 무료) + 녹음실(월 8시간 무료)
3. **전주 영화 후반제작지원 사업 지원**: 연 2회 공모, 색보정·사운드·DCP 현물 지원
4. **학생/프리랜서 협업**: 필름메이커스 커뮤니티를 통한 상호 포트폴리오 교환
5. **DCP-o-matic (무료)**: 오픈소스 DCP 생성 도구, 극장 테스트 필수

## 9. Pro Mode 연동 시사점

Cinema On Air의 Pro Mode가 후반작업 인프라 정보를 활용할 수 있는 방안:

### 9.1 후반작업 스펙 메타데이터 태깅

- 영화 업로드 시 후반작업 파이프라인 메타데이터 자동 기록:
  - 편집 소프트웨어 (Premiere / Resolve / Final Cut)
  - 색보정 수행 여부 및 도구 (DaVinci Resolve DI / 전문 스튜디오 DI)
  - 사운드 믹싱 방식 (자체 / 전문 스튜디오)
  - DCP 여부 및 규격 (2K Flat / 4K Scope 등)
- **제작 파이프라인 투명화**: "이 영화가 어떤 과정을 거쳐 완성되었는가"를 관객과 동료 감독에게 공유

### 9.2 필터 및 검색 기능

- **"DaVinci Resolve로 제작한 독립영화"** 필터: 같은 도구를 사용하는 감독들의 결과물 비교·학습
- **"전주 후반제작지원 사업 선정작"** 태그: 공공 지원의 성과 가시화
- **후반작업 수준별 필터**: "풀 DI 색보정 완료" / "5.1ch 사운드 믹싱" / "DCP 상영 가능" 등

### 9.3 서비스 정보 연동

- DCP 제작 서비스 정보 링크: 영화제 출품 준비 시 DCP 변환 서비스 바로 연결
- 공공 지원 사업 공모 일정 알림: 전주 후반제작지원 사업 등 공모 시작 시 Pro Mode 사용자에게 푸시 알림
- **후반작업 비용 견적 도구**: 작품 러닝타임·해상도·사운드 채널 입력 시 예상 후반작업 비용 범위 제시

### 9.4 커뮤니티 기능

- 후반작업 크레딧 공유: 색보정사, 사운드 믹서 등 프리랜서 크레딧을 프로필로 연결
- "이 컬러리스트와 함께 작업한 영화 목록" 기능: 포트폴리오 기반 매칭
