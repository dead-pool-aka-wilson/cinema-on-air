# 법적/행정: 한국 독립영화/단편영화 제작 인프라

## 1. 개요

한국에서 독립영화/단편영화를 제작하려면 창작 역량 외에도 다양한 법적·행정적 절차를 거쳐야 한다. 영화제작업 신고, 촬영 허가, 저작권 처리, 등급 심의, 보험 가입, 세무 처리, 계약서 작성 등이 핵심 영역이다.

> **면책 고지**: 이 문서는 정보 안내 목적이며, 법률 자문이 아니다. 구체적 법적 판단은 반드시 변호사 등 전문가와 상담할 것.

**핵심 장벽**: 영진위 지원사업 신청 시 "영화연출경력 + 영화제작업신고증"이 필수 → 첫 영화를 만드는 신인 감독에게는 진입 장벽이 높다.

## 2. 영화제작업 신고 절차

### 관할 기관

- **영화진흥위원회** (kofic.or.kr) — 영화산업 진흥 총괄, 독립영화 인정, 등급분류 면제 추천
- **관할 시·군·구청 문화 부서** — 실제 영화업 신고 접수처 (2009년 법 개정 이후 영진위 → 지자체로 이관)

### 법적 근거

- 「영화 및 비디오물의 진흥에 관한 법률」 제26조: 영화업자가 되려는 자는 시장·군수·구청장에게 신고 의무
- 미신고 시 **1,000만 원 이하 과태료** 부과 가능

### 필요 서류

| 서류 | 비고 |
|------|------|
| 영화업 신고서 (별지 제1호의2서식) | 구청 홈페이지 또는 정부24에서 다운로드 |
| 사업자등록증 | 세무서 발급, 업종코드 영화제작업 |
| (법인) 등기사항전부증명서 | 말소사항 포함, 개인사업자는 불필요 |
| 임대차계약서 | 공유오피스 전대차 시 전대차동의서 필수 |
| 대표자 신분증 사본 | 대리인 방문 시 인감증명서+위임장 추가 |

### 신고 방법 및 처리

- **온라인**: 정부24 (gov.kr) — 법인은 대표자 개인인증으로 접속
- **오프라인**: 관할 구청 문화 부서 방문 접수
- **수수료**: 신규 30,000원, 변경 15,000원, 재교부 7,500원
- **처리기간**: 약 5일 [UNVERIFIED — 지자체별 상이]

### 개인사업자 vs 법인

- 최소 자본금 요건 없음 (법적 제한 없음)
- 소규모 독립영화: 개인사업자로도 충분
- 투자 유치·장기 운영 계획 시: 법인(주식회사/유한회사) 권장

## 3. 촬영 허가 절차

한국에서 공공장소 촬영은 장소 유형별로 관할 기관이 다르다.

### 장소별 담당 기관

| 촬영 장소 | 담당 기관/절차 |
|-----------|---------------|
| 도로·차도 | 관할 경찰서 (도로사용 허가) + 구청 도로관리과 |
| 공원 (서울시 관리) | 서울시 공원녹지과 — 촬영일 7일 전 신청, 이용료 시간당 14,300원 (초과 7,150원) |
| 지하철 | 노선별 운영기관 (서울교통공사, 코레일 등) — 서울영상위원회에서 절차 안내 |
| 교육시설 | 해당 학교 행정실 — 운영 외 시간 촬영 원칙 |
| 군사시설 | 국방부 협조 필요 |
| 사유지·상업시설 | 소유자/관리자 직접 협의 |

### 서울영상위원회 (Seoul Film Commission)

- **웹사이트**: seoulfc.or.kr
- 서울 내 촬영 지원 신청, 로케이션 DB 검색, 촬영 매뉴얼 제공
- **촬영지원 절차**: 로케이션 지원신청서 제출 → 담당자 배정 → 지원범위 조율 → 촬영 허가 요청 → 촬영 지원
- 전국 통합 DB: **필름코리아** (filmcorea.or.kr) — 전국영상위원회 통합 촬영지원 신청 포털

### 기타 지역 영상위원회

- 부산영상위원회 (BFC), 경기영상위원회, 전주영상위원회 등 각 지역별 운영
- 지역 촬영 시 해당 지역 영상위원회를 통해 촬영 허가 조율 가능

## 4. 초상권 및 출연 동의서

### 법적 기반

- 한국 민법상 초상권은 인격권의 일종으로 보호 (명문 규정은 없으나 판례로 확립)
- 촬영 대상자의 동의 없이 초상을 사용하면 민사상 손해배상 청구 가능

### 동의서 필수 항목

- 촬영 대상자 인적사항 (이름, 연락처)
- 촬영 목적 및 사용 범위 (영화제 출품, 온라인 공개, 방송 등)
- 사용 기간 (특정 기간 또는 영구)
- 초상 사용료 (있는 경우)
- 동의 철회 조건
- 미성년자의 경우 **법정대리인(부모) 동의** 필수

### 실무 팁

- 엑스트라 포함 모든 출연자에게 반드시 서면 동의서 확보
- 거리 촬영 시 불특정 다수가 포함되는 경우: 촬영 고지문 게시 + 편집 시 비식별 처리 권장
- 문화체육관광부 표준계약서 양식 활용 가능

## 5. 저작권 — 배경음악 및 기존 저작물 사용

### 음악 저작권의 3가지 권리

영화에 기존 음악을 삽입하려면 세 가지 권리를 모두 확보해야 한다:

1. **저작권** (작곡가·작사가) — KOMCA 관리
2. **저작인접권 — 실연권** (가수·연주자) — 한국음악실연자연합회(FOM) 관리
3. **저작인접권 — 음반제작권** (음반사) — 해당 레이블 직접 협의

### KOMCA (한국음악저작권협회)

- **웹사이트**: komca.or.kr
- 1964년 설립, 문화체육관광부 허가 비영리단체
- 작곡가·작사가의 저작권을 위탁 관리, 사용료 징수 및 분배
- **영화 음악 사용료 신청**: KOMCA에 영상복제 사용 허가 신청 → 사용료 납부
- 해외 곡의 경우: KOMCA에서 국내 퍼블리셔 연락처 안내 → 퍼블리셔를 통해 해외 원저작자 승인 필요

### 독립영화/학생영화의 저작권료

- 학생 단편영화·독립영화의 경우 저작권료 감면 또는 무료 협의가 가능한 경우 있음 [UNVERIFIED — 곡별·저작권자별 상이]
- 대안: **로열티 프리 음악 라이선스 서비스** 활용
  - 뮤직플러그 (musicplug.co.kr) — 경기영상위원회 협약, 곡당 27,500원부터
  - Artlist, Epidemic Sound 등 해외 서비스

### 기존 저작물 (소설, 사진, 미술 등) 사용

- 2차적저작물 작성권: 원저작자의 허락 필요 (저작권법 제22조)
- 공정이용(Fair Use) 조항: 저작권법 제35조의5 — 제한적 적용, 상업적 목적 시 인정 어려움

## 6. 영상물 등급 심의

### 관할 기관

- **영상물등급위원회** (kmrb.or.kr) — 부산 해운대구 센텀시티 소재
- 「영화 및 비디오물의 진흥에 관한 법률」 제29조에 근거

### 등급 분류 체계

| 등급 | 설명 |
|------|------|
| 전체관람가 | 모든 연령 관람 가능 |
| 12세 이상 관람가 | 12세 이상 관람 가능 |
| 15세 이상 관람가 | 15세 이상 관람 가능 |
| 청소년 관람불가 | 청소년 관람 불가 |
| 제한상영가 | 상영·광고에 일정 제한 |

### 독립영화/단편영화 등급분류 면제 조건

법 제29조 제1항 단서에 따라 다음의 경우 등급분류를 받지 않아도 된다:

1. **소형영화·단편영화**: 대가를 받지 않고, 특정 장소에서, 청소년이 포함되지 않은 특정인에게만 상영하는 경우
2. **영화진흥위원회 추천 영화제**에서 상영하는 영화 — 영진위에서 "등급분류면제추천" 업무 수행 (kofic.or.kr)
3. **국제 문화교류 목적** 상영 등 문화체육관광부장관이 인정하는 경우

### 실무 시사점

- 독립영화제·단편영화제 출품만 할 경우: 영진위 등급분류면제추천으로 심의 없이 상영 가능
- 극장 개봉 또는 온라인 유료 공개 시: 반드시 영상물등급위원회에서 등급분류 필요
- 예술영화 인정 시 등급분류 수수료 환불 제도 존재 [UNVERIFIED — 연도별 정책 변동]

## 7. 촬영 보험

### 주요 보험 유형

| 보험 종류 | 보장 내용 |
|-----------|-----------|
| **상해보험** | 실연자·스태프 상해, 후유장애, 사망 시 의료비·보험금 |
| **책임보험** (배상책임) | 제작 과정에서 제3자 신체·재산 손해 배상 |
| **손해보험** | 소품, 세트, 의상, 촬영 장비 파손·도난 |
| **E&O보험** | 저작권·상표권 침해 주장 시 소송 비용 (글로벌 배급 시 사실상 필수) |
| **완성보증보험** | 예산 초과 등으로 프로그램 완성 실패 시 리스크 보장 |

### 의무 가입 보험

- **4대 보험**: 근로자성이 있는 스태프에 대해 국민연금, 건강보험, 고용보험, 산재보험 가입 의무
- **예술인 고용보험**: 월 50만 원 이상 또는 1개월 미만 단기 계약의 문화예술용역 제공자에게 당연 적용 (고용보험법 제77조의2)
- **한국콘텐츠진흥원** + **신용보증기금**: 문화산업 완성보증사업 운영 (문화산업진흥 기본법 제10조의2)

### 독립영화 현실

- 저예산 독립영화는 상해보험 + 책임보험 정도만 가입하는 경우가 다수
- 영진위 등 공적 지원금 수령 시 보험 가입 증빙 요구되는 경우 있음
- 촬영 보험 전문 중개: 미디어제작보험 취급 보험대리점 활용

## 8. 세무/회계 — 프리랜서 감독의 세무 처리

### 기본 구조

- 프리랜서 영화감독·스태프 = 세법상 **사업소득자**
- 수입 지급 시 **3.3% 원천징수** (소득세 3% + 지방소득세 0.3%)
- 이 3.3%는 "예납"일 뿐, 다음 해 5월 **종합소득세 확정신고**로 정산

### 종합소득세 신고

- **신고 기간**: 매년 5월 1일 ~ 5월 31일
- **신고 방법**: 국세청 홈택스 (hometax.go.kr) 또는 관할 세무서 방문
- **필요경비 처리**: 장비 구매비, 소프트웨어, 촬영 경비, 통신비 등 → 과세 대상 소득 감소
- **경비 방식**: 단순경비율 (소규모) vs 간편장부/복식부기 (매출 규모에 따라)

### 제작비 세무 처리

- 영화 제작비는 사업자등록 후 적격 증빙(세금계산서, 카드전표, 현금영수증)으로 비용 처리
- 스태프·배우 인건비 지급 시: 원천징수 후 세무서 신고 의무
- 미신고·과소신고 시 가산세: 무신고 20%, 납부지연 일 0.022%

## 9. 계약서 가이드 (배우, 스태프, 장소)

> **주의**: 이 섹션은 계약서 체크리스트 안내이며, 법적 효력이 있는 계약서 템플릿을 제공하지 않는다.

### 표준계약서 출처

- **문화체육관광부**: 대중문화예술인 방송출연 표준계약서, 영화산업 스태프 표준계약서 등 공개
- **예스폼** (yesform.com): 영화산업 스태프 표준계약서 양식 (유료)
- **모두싸인** (modusign.co.kr): 표준계약서 양식 + 전자서명 서비스

### 배우 계약서 핵심 항목

- 출연료 및 지급 일정
- 촬영 기간·일정 (일일 최대 촬영시간: 18시간 초과 금지)
- 초상권 사용 범위·기간
- 2차 사용 (VOD, 해외 배급 등) 시 추가 보상
- 안전 보장 및 상해보험 가입 의무
- 계약 해지 시 위약금

### 스태프 계약서 핵심 항목

- 소속 부서·직위·담당 업무 (프리/프로덕션/포스트 단계별 명시)
- 임금 산정 기준 (월급제 vs 회차제)
- 초과근무수당 계산 방법
- 4대 보험 가입 여부
- 저작권 귀속 (제작사 귀속 vs 공동 귀속)

### 장소 사용 계약 핵심 항목

- 사용 일시·기간
- 사용료 및 보증금
- 원상복구 의무 및 손해배상 범위
- 촬영 중 소음·진동 등에 대한 민원 대응 책임

## 10. Pro Mode 연동 시사점

Cinema On Air의 Pro Mode는 독립영화 제작자를 위한 실전 가이드 역할을 해야 한다. 법적/행정 영역에서의 연동 포인트:

### 영화제작업 신고 가이드

- 영진위 신고 절차 스텝 바이 스텝 안내 + 정부24 링크 직접 연결
- 개인사업자 vs 법인 선택 의사결정 트리
- 필요 서류 체크리스트 (사업자등록증, 임대차계약서 등)

### 촬영 허가 체크리스트

- 촬영 장소 유형 선택 → 담당 부처 자동 안내 (도로/공원/지하철/학교 등)
- 서울영상위원회 촬영지원 신청 링크 (seoulfc.or.kr)
- 필름코리아 전국 촬영지원 포털 연동

### 표준 동의서/계약서 연결

- 문화체육관광부 표준계약서 다운로드 페이지 링크
- 초상권 사용동의서 템플릿 안내 (공공 도메인 양식 연결)
- 영화산업 스태프 표준계약서 양식 안내

### 저작권 클리어런스 워크플로우

- KOMCA 음악 저작권 조회 시스템 연동 안내
- 로열티 프리 음악 서비스 (뮤직플러그 등) 비교표
- 저작권 확보 필요 여부 판단 플로우차트

### 세무/보험 체크리스트

- 프리랜서 3.3% 원천징수 → 종합소득세 신고 리마인더
- 촬영 보험 가입 체크리스트 (상해+책임 최소 권장)
- 예술인 고용보험 적용 여부 자가 진단
