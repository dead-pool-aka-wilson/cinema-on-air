# H5 배급사 인터뷰 가이드 — B2B 데이터 가치 검증

## 1. 개요

| 항목 | 내용 |
|------|------|
| **목적** | 배급사/영화제 프로그래머가 플랫폼 관객 데이터를 감독 발굴에 활용할 의향이 있는지 검증 |
| **대상** | 독립영화 배급사 관계자, 영화제 프로그래머 3~5인 |
| **형식** | 반구조화 인터뷰 (Semi-structured Interview), 30~45분 |
| **성공 기준** | 5명 중 **3명 이상**이 "플랫폼 데이터를 배급/프로그래밍에 참고하겠다" 답변 |

---

## 2. 인터뷰 대상 프로필

### 2.1 우선 섭외 대상

| 유형 | 기관 예시 | 섭외 방법 |
|------|----------|---------|
| **독립영화 배급사** | 인디스토리, 시네마달, 영화사 진진 | 공식 이메일, LinkedIn |
| **영화제 프로그래머** | 부산국제단편영화제, 서울독립영화제, 전주국제영화제 | 공식 이메일, 영화제 홈페이지 |
| **OTT 콘텐츠 담당** | 왓챠, 네이버 시리즈온 | 공식 이메일, LinkedIn |
| **영화 투자사** | 독립영화 투자 경험 있는 소규모 투자사 | 네트워크 소개 |

### 2.2 섭외 이메일 템플릿

```
제목: [Cinema on Air] 독립영화 플랫폼 관련 인터뷰 요청 (30분)

안녕하세요, [이름]님.

저는 독립영화 감독들이 직접 작품을 업로드하고, 관객 지표(조회수, 완주율)를 통해
배급사가 신인 감독을 발굴하는 플랫폼 'Cinema on Air'를 준비 중입니다.

플랫폼 개발 전, 배급사/영화제 관계자분들의 실제 니즈를 파악하고자 합니다.
30분 정도의 인터뷰를 요청드려도 될까요?

인터뷰 내용은 플랫폼 기획에만 활용되며, 개인 정보는 공개되지 않습니다.

감사합니다.
[이름] 드림
```

## 3. 인터뷰 질문 목록 (15개)

### 섹션 A: 현재 감독 발굴 프로세스 (5개)

**A1.** 현재 신인 감독이나 새로운 작품을 발굴하는 주요 채널은 무엇인가요?
> *탐색 목적: 현재 발굴 방식의 한계 파악*

**A2.** 영화제 출품작 외에 작품을 발굴하는 방법이 있으신가요?
> *탐색 목적: 영화제 외 채널 활용 여부*

**A3.** 신인 감독의 작품을 선택할 때 가장 중요하게 보는 기준은 무엇인가요?
> *탐색 목적: 의사결정 기준 파악*

**A4.** 현재 감독 발굴 프로세스에서 가장 불편하거나 아쉬운 점은 무엇인가요?
> *탐색 목적: 페인 포인트 발굴*

**A5.** 영화제에 선정되지 못한 작품 중 나중에 주목받은 사례를 경험하신 적 있으신가요?
> *탐색 목적: 영화제 필터링의 한계 인식 여부*

---

### 섹션 B: 데이터 활용 의향 (5개)

**B1.** 만약 플랫폼에서 특정 단편영화의 조회수, 완주율, 관객 리뷰 데이터를 제공한다면, 이를 감독 발굴 의사결정에 참고하실 의향이 있으신가요?

*(1점: 전혀 없다 ~ 5점: 반드시 활용하겠다)*

**B2.** 어떤 종류의 데이터가 가장 유용할 것 같으신가요? (복수 선택 가능)
- [ ] 총 조회수
- [ ] 완주율 (영화를 끝까지 본 비율)
- [ ] 관객 리뷰/평점
- [ ] 시청자 연령/지역 분포
- [ ] 재시청률
- [ ] 소셜 공유 횟수
- [ ] 기타: ___________

**B3.** 데이터를 신뢰하기 위해 필요한 조건은 무엇인가요?
> *탐색 목적: 데이터 신뢰도 요건 파악*

**B4.** 플랫폼 데이터를 활용한다면, 현재 발굴 프로세스의 어느 단계에서 활용하실 것 같으신가요?
> *탐색 목적: 실제 활용 시나리오 파악*

**B5.** 데이터 기반 발굴 방식이 기존 방식(영화제, 네트워크)을 대체할 수 있다고 생각하시나요, 아니면 보완적으로 활용하실 것 같으신가요?
> *탐색 목적: 대체 vs 보완 포지셔닝 파악*

---

### 섹션 C: 플랫폼 가치 제안 검증 (5개)

**C1.** Cinema on Air 같은 플랫폼이 있다면, 정기적으로 방문하여 신인 감독을 탐색하실 의향이 있으신가요?

*(1점: 전혀 없다 ~ 5점: 반드시 방문하겠다)*

**C2.** 플랫폼이 제공해야 할 가장 중요한 기능은 무엇이라고 생각하시나요?
> *탐색 목적: B2B 핵심 기능 우선순위 파악*

**C3.** 플랫폼을 통해 발굴한 감독과 계약하거나 협업하는 것에 대해 어떻게 생각하시나요?
> *탐색 목적: 실제 비즈니스 연결 가능성 파악*

**C4.** 이런 플랫폼이 한국 독립영화 생태계에 어떤 영향을 미칠 것 같으신가요?
> *탐색 목적: 생태계 변화에 대한 인식 파악*

**C5.** 마지막으로, 이 플랫폼이 성공하기 위해 가장 중요한 것은 무엇이라고 생각하시나요?
> *탐색 목적: 전문가 관점의 성공 요인 파악*

## 4. 인터뷰 진행 가이드

### 4.1 인터뷰 전 준비
- 인터뷰 대상자의 기관/역할 사전 조사
- 녹음 동의 여부 확인 (동의 시 녹음, 미동의 시 메모)
- 플랫폼 개요 1페이지 자료 준비 (인터뷰 시작 전 공유)

### 4.2 인터뷰 진행 순서
1. 자기소개 및 플랫폼 개요 설명 (5분)
2. 섹션 A: 현재 발굴 프로세스 (10분)
3. 섹션 B: 데이터 활용 의향 (10분)
4. 섹션 C: 플랫폼 가치 제안 (10분)
5. 자유 의견 및 마무리 (5분)

### 4.3 인터뷰 후 처리
- 24시간 내 메모 정리
- 핵심 인사이트 3가지 추출
- B1, C1 점수 기록 (성공 기준 측정용)

## 5. 성공 기준 및 측정

| 지표 | 성공 기준 | 측정 방법 |
|------|----------|---------|
| **데이터 활용 의향 (B1)** | 5명 중 **3명 이상** 4~5점 | B1 응답 집계 |
| **플랫폼 방문 의향 (C1)** | 5명 중 **3명 이상** 4~5점 | C1 응답 집계 |
| **정성적 인사이트** | 공통 페인 포인트 3개 이상 발굴 | 인터뷰 메모 분석 |
