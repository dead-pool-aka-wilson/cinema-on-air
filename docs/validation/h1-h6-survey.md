# H1 + H6 설문 문항 설계 — 공급 의지 & 등록비 수용도

## 1. 설문 개요

| 항목 | 내용 |
|------|------|
| **목적** | 독립영화 감독들이 Cinema on Air 플랫폼에 작품을 올릴 의향이 있는지, 소액 등록비를 수용할 의향이 있는지 측정 |
| **대상** | 단편영화를 1편 이상 제작한 경험이 있는 감독/크리에이터 |
| **예상 소요 시간** | 5~7분 |
| **목표 응답 수** | 최소 50명 (유효 응답 기준) |
| **도구** | Google Forms |

---

## 2. H1 설문 문항 — 공급 의지 (Supply Willingness)

### [스크리닝] 기본 자격 확인

**Q1.** 단편영화(30분 이하)를 직접 연출하거나 제작한 경험이 있으신가요?
- [ ] 예, 1편 이상 완성했습니다
- [ ] 현재 제작 중입니다
- [ ] 아직 없지만 계획 중입니다
- [ ] 없습니다 *(→ 설문 종료)*

**Q2.** 가장 최근에 완성한 단편영화는 언제 제작하셨나요?
- [ ] 2024년 이후
- [ ] 2022~2023년
- [ ] 2020~2021년
- [ ] 2019년 이전

---

### [핵심] 플랫폼 업로드 의향

**Q3.** 현재 완성된 단편영화를 어떤 방식으로 공개하고 계신가요? (복수 선택 가능)
- [ ] 영화제 출품 (국내)
- [ ] 영화제 출품 (해외)
- [ ] 유튜브/비메오 업로드
- [ ] SNS (인스타그램, 틱톡 등)
- [ ] 지인에게만 공유
- [ ] 공개하지 않고 있음
- [ ] 기타: ___________

**Q4.** 현재 배급/공개 방식에서 가장 불편하거나 아쉬운 점은 무엇인가요? (오픈형)
> 자유롭게 작성해 주세요.

**Q5.** "Cinema on Air"는 단편영화 감독이 직접 작품을 업로드하고, 관객이 스트리밍으로 감상하며, 배급사가 관객 지표(조회수, 완주율)를 보고 감독을 발굴하는 플랫폼입니다. 이 플랫폼에 본인의 작품을 올릴 의향이 있으신가요?

*(1점: 전혀 없다 ~ 5점: 반드시 올리겠다)*

| 1점 | 2점 | 3점 | 4점 | 5점 |
|-----|-----|-----|-----|-----|
| 전혀 없다 | 없다 | 보통 | 있다 | 반드시 올리겠다 |

**Q6.** Q5에서 선택한 이유를 간략히 설명해 주세요. (오픈형)
> 자유롭게 작성해 주세요.

**Q7.** 플랫폼에 작품을 올리기 위해 가장 중요한 조건은 무엇인가요? (최대 3개 선택)
- [ ] 내 작품의 저작권이 완전히 보호됨 (비독점 라이선스)
- [ ] 관객 반응 데이터(조회수, 완주율, 리뷰)를 직접 확인 가능
- [ ] 배급사/영화제 관계자가 실제로 플랫폼을 이용함
- [ ] 다른 플랫폼(유튜브, 영화제)에 동시 출품 가능
- [ ] 플랫폼의 큐레이션/추천 시스템이 내 작품을 노출시켜 줌
- [ ] 수익 창출 가능성 (유료 상영, 수익 분배)
- [ ] 기타: ___________

**Q8.** 현재 영화제 출품 외에 작품을 알릴 수 있는 채널이 충분하다고 생각하시나요?
- [ ] 충분하다
- [ ] 부족하지만 그럭저럭 괜찮다
- [ ] 매우 부족하다
- [ ] 아예 없다고 느낀다

**Q9.** 영화제에 출품했지만 선정되지 못한 작품이 있으신가요?
- [ ] 예, 있습니다
- [ ] 아니오, 없습니다
- [ ] 영화제에 출품한 적 없습니다

**Q10.** 만약 Cinema on Air 같은 플랫폼이 있었다면, 영화제 미선정 작품을 올렸을 것 같으신가요?
- [ ] 예, 올렸을 것입니다
- [ ] 아마 올렸을 것입니다
- [ ] 잘 모르겠습니다
- [ ] 아마 올리지 않았을 것입니다
- [ ] 아니오, 올리지 않았을 것입니다

## 3. H6 설문 문항 — 등록비 수용도 (Monetization Acceptance)

**Q11.** 플랫폼에 작품을 등록할 때 스팸 방지 및 품질 관리를 위해 소액의 등록비를 부과한다면 어떻게 생각하시나요?
- [ ] 합리적이다, 기꺼이 낼 의향이 있다
- [ ] 이해는 하지만 부담스럽다
- [ ] 반대한다, 무료여야 한다
- [ ] 조건에 따라 다르다

**Q12.** 작품 1편당 등록비로 수용 가능한 금액은 얼마인가요?
- [ ] 무료여야 한다
- [ ] 5,000원 이하
- [ ] 5,000원 ~ 10,000원
- [ ] 10,000원 ~ 20,000원
- [ ] 20,000원 ~ 30,000원
- [ ] 30,000원 이상도 괜찮다

**Q13.** 등록비 대신 원하는 혜택이 있다면 무엇인가요? (오픈형)
> 예: 관객 데이터 리포트, 배급사 노출 보장, 큐레이션 선정 기회 등

**Q14.** 등록비를 내더라도 플랫폼을 이용하겠다는 조건은 무엇인가요? (복수 선택 가능)
- [ ] 실제 배급사/영화제 관계자가 플랫폼을 이용한다는 증거
- [ ] 관객 데이터 리포트 제공 (조회수, 완주율, 지역별 통계)
- [ ] 큐레이션 섹션에 선정될 기회
- [ ] 다른 감독들의 네트워킹 기회
- [ ] 플랫폼의 마케팅 지원
- [ ] 기타: ___________

**Q15.** 마지막으로, Cinema on Air에 대해 하고 싶은 말씀이 있으시면 자유롭게 작성해 주세요. (오픈형)
> 기대하는 점, 우려되는 점, 제안 사항 등 무엇이든 환영합니다.

## 6. 설문 도구 설정 가이드 (Google Forms)

```
설문 제목: Cinema on Air — 단편영화 플랫폼 사전 조사
설문 설명: 안녕하세요. 독립영화 감독들을 위한 새로운 플랫폼 'Cinema on Air'를 준비 중입니다.
           여러분의 소중한 의견을 듣고 싶습니다. 5~7분 정도 소요됩니다.

섹션 1: 기본 정보 (Q1~Q2) — 스크리닝
섹션 2: 현재 배급 현황 (Q3~Q4)
섹션 3: 플랫폼 업로드 의향 (Q5~Q10) — 핵심
섹션 4: 등록비 수용도 (Q11~Q14) — 핵심
섹션 5: 자유 의견 (Q15)

응답 수집 후 처리:
- Q5 4~5점 응답자 → H1 성공 카운트
- Q12 5,000원+ 응답자 → H6 성공 카운트
- Q4, Q6, Q13, Q15 → 정성적 인사이트 분석
```
