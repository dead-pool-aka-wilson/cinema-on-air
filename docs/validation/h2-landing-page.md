# H2 랜딩페이지 콘텐츠 초안 — 수요 측면 검증

## 1. 개요

| 항목 | 내용 |
|------|------|
| **목적** | 독립영화 특화 플랫폼에 대한 관객 수요 측정 (이메일 등록 전환율) |
| **대상** | 단편영화에 관심 있는 시네필, 영화 애호가 |
| **성공 기준** | 방문자 중 **5% 이상** 이메일 등록 |
| **도구** | Framer / Notion / Carrd (코딩 없이 제작 가능) |

---

## 2. 랜딩페이지 구조

### 섹션 1: 히어로 (Hero)

**헤드라인 (A안):**
> 영화제가 외면한 단편영화들이 여기 있습니다

**헤드라인 (B안 — A/B 테스트용):**
> 유튜브에서 찾을 수 없는 단편영화를 발견하세요

**서브헤드라인:**
> 매년 수천 편의 단편영화가 만들어지지만, 98%는 영화제 문턱을 넘지 못하고 사라집니다.
> Cinema on Air는 그 영화들에게 관객을 연결합니다.

**CTA 버튼:**
> 🎬 오픈 알림 받기 (무료)

**보조 텍스트:**
> 스팸 없음. 언제든 구독 취소 가능.

---

### 섹션 2: 문제 제기 (Problem)

**제목:** 단편영화를 보고 싶은데, 어디서 봐야 할지 모르겠다면?

**내용:**
- 유튜브에는 단편영화가 묻혀 있고
- 영화제는 1년에 한 번, 선정된 작품만 볼 수 있고
- 왓챠·넷플릭스에는 단편영화 섹션이 없다

> "졸업작을 3년 만에 다시 편집했는데 보여줄 곳이 없어 커뮤니티에 올린다"
> — 실제 독립영화 감독의 말

---

### 섹션 3: 솔루션 (Solution)

**제목:** Cinema on Air — 단편영화를 위한 Steam

**3가지 핵심 가치:**

1. **발견** — 태그, 무드, 감독 필모그래피로 내 취향의 단편영화를 찾는다
2. **아카이빙** — 영화제 미선정작부터 졸업작까지, 모든 단편영화의 영구 보관소
3. **연결** — 관객의 반응이 감독의 커리어를 만든다

---

### 섹션 4: 기능 미리보기 (Features Preview)

| 기능 | 설명 |
|------|------|
| 🏷️ **태그 기반 검색** | 장르, 무드, 테마, 런닝타임으로 정확하게 |
| 🎬 **감독 필모그래피** | 나홍진의 첫 단편부터 최신작까지 한눈에 |
| 📊 **관객 지표** | 조회수, 완주율, 리뷰로 작품성을 증명 |
| 🏆 **큐레이션** | 영화제, 평론가, 시네필이 선별한 리스트 |

---

### 섹션 5: 사회적 증거 (Social Proof) — 런칭 전 대체 콘텐츠

**제목:** 이런 분들을 위해 만들었습니다

> "영화제 선정작만 보다 보면 매년 같은 연출, 같은 소재, 같은 연기다."
> — 시네필 커뮤니티 반응

> "단편영화 DB 업데이트나 빨리 하라. 영화제 간 단편들 정보가 없다."
> — 관객 커뮤니티 반응

> "보고 쓰고 찍고 **보이자**"
> — 독립영화 감독의 말

---

### 섹션 6: CTA (Call to Action)

**제목:** 오픈 알림을 받으세요

**설명:**
> Cinema on Air는 현재 준비 중입니다.
> 이메일을 남겨주시면 오픈 소식을 가장 먼저 알려드립니다.

**입력 필드:**
- 이메일 주소 (필수)
- 이름 (선택)
- 나는: [ ] 영화 보는 것을 좋아하는 관객 / [ ] 단편영화를 만드는 감독

**CTA 버튼:**
> 🎬 오픈 알림 받기

**완료 메시지:**
> 감사합니다! 오픈 소식을 가장 먼저 알려드릴게요. 🎬

## 3. A/B 테스트 변형안

### A안 — 감성 소구 (Emotional Appeal)
- 헤드라인: "영화제가 외면한 단편영화들이 여기 있습니다"
- 이미지: 어두운 영화관 + 스크린 빛
- 톤: 시네마틱, 감성적

### B안 — 기능 소구 (Functional Appeal)
- 헤드라인: "유튜브에서 찾을 수 없는 단편영화를 발견하세요"
- 이미지: 깔끔한 영화 목록 UI 목업
- 톤: 명확하고 실용적

**측정 지표:** 각 변형안의 이메일 등록 전환율 비교

## 4. 배포 채널

| 채널 | 방법 | 예상 도달 수 |
|------|------|-------------|
| **DC Inside 누벨바그 갤러리** | 게시글 작성 (서비스 소개 + 링크) | 500~1,000 뷰 |
| **DC Inside 영화 갤러리** | 게시글 작성 | 300~500 뷰 |
| **트위터/X 영화 커뮤니티** | 스레드 작성 (#단편영화 #독립영화) | 200~500 뷰 |
| **인스타그램** | 릴스/스토리 (#단편영화 #인디영화) | 100~300 뷰 |
| **네이버 카페 (영화 관련)** | 게시글 작성 | 100~200 뷰 |

## 5. 성공 기준

| 지표 | 성공 기준 | 측정 방법 |
|------|----------|---------|
| **이메일 등록 전환율** | 방문자 **5% 이상** | Google Analytics + 폼 응답 수 |
| **방문자 수** | 최소 **200명** (전환율 측정 유의미성) | Google Analytics |
| **이메일 등록 수** | 최소 **10명** | 폼 응답 수 |
| **관객 vs 감독 비율** | 관객 60%+ | 폼 응답 분석 |
